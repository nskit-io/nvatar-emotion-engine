# NVatar Emotion Engine

**9次元感情トラッキング + MBTI連続スペクトラム + decay/commitによる性格進化 + 3層記憶圧縮 + ユーザー性格推論 — AIキャラクターのための統合コグニティブエンジン。**

> [NVatar](https://github.com/nskit-io/nvatar)プロジェクトより — ローカルハードウェアで完全に動作するAIアバターチャットシステム。

---

## なぜ作ったのか

今日のLLMキャラクターはステートレスです。忘れ、成長せず、感情もない。メモリライブラリは存在しますが（Letta、Mem0）、**感情 + 性格進化 + 記憶**を統合したシステムはありません。

このエンジンがAIキャラクターに与えるもの：
- **感情**: 会話中に自然に変動し、ベースラインに減衰（好奇心を含む9次元）
- **MBTIスペクトラム**: 連続的0-100スケールで確率的表現とともに進化 — 固定タイプではない
- **性格**: 週単位のインタラクションで進化 — 独自のdecay/commitメカニズム
- **記憶**: 生メッセージ → 要約 → キーワードに圧縮、人間の記憶が薄れるように
- **ユーザー理解** — アバターが会話パターンからあなたの性格を推論し、スタイルを調整

## アーキテクチャ

```
ユーザーメッセージ
    │
    ├─→ 感情サービス ────→ 9次元感情ベクトル更新
    │                       （キーワードトリガー、自然減衰）
    │
    ├─→ 記憶サービス ────→ L1生 → L2イベント → L3キーワード
    │                       （自動圧縮 + L2→L3自動遷移）
    │
    ├─→ 性格進化 ───────→ 特性ドリフト + decay/commit
    │                       （保留デルタ → 時間減衰 → コミット）
    │
    ├─→ MBTIアナライザー ─→ 連続スペクトラム進化
    │                       （ブラインド分析 → デルタ → 加重適用）
    │
    └─→ ユーザー推論 ───→ ユーザーメッセージから性格を推論
                            （圧縮サイクルを通じて蓄積）
```

## 感情システム — 9次元

感情分析ではありません。分類でもありません。会話を超えて**持続する連続的な感情状態**です。

| 次元 | 傾向 |
|------|------|
| 喜び | ポジティブなインタラクションで上昇 |
| 悲しみ | 喪失/失望のトピックに反応 |
| 怒り | 対立で急上昇、最速で減衰 |
| 不安 | 不確実性で蓄積 |
| 愛情 | 信頼とともにゆっくり成長 |
| 興奮 | 新奇なものに急上昇 |
| 退屈 | 繰り返しで上昇 |
| 信頼 | 最もゆっくり変化する次元 |
| **好奇心** | **新しいトピックで上昇 — フォローアップ質問とより深い探求をトリガー** |

**メカニクス：**
- キーワードが比例的な感情変化をトリガー
- 複数ターンにわたりベースラインへ自然減衰
- 有意に高まった感情のみプロンプトに反映
- 各メッセージに`emotion_snapshot`を保存し、遡及的分析が可能

## 性格進化 — Decay/Commit

**このメカニズムのオープンソースの先行事例は存在しません。**

8つの性格特性が3段階パイプラインで進化：

**ステージ1**: メッセージごとに特性デルタを蓄積 — 設定可能な上限で極端な変動を防止。

**ステージ2 — 時間減衰**
最後の会話からの経過時間に基づく指数減衰 — 最近の会話は強く影響し、古い会話は自然に薄れる。

**ステージ3 — 圧縮コミット**: 保留中のデルタに減衰係数を適用してコミット、設定可能な上限付き。

**抵抗曲線：** 極端な値はさらなる変化に抵抗する — 実際の人間のように。

すべてのコミットは分析のためにログ記録されます。

## MBTI連続スペクトラム

固定された性格タイプ（INTJ、ENFP...）とは異なり、NVatarは4つのMBTI次元にわたる**連続的0-100スペクトラム**を使用します。アバターは一つのタイプに固定されません — 会話を通じて変化する生きた性格です。

| 次元 | 0 | 50（境界） | 100 |
|------|---|-----------|-----|
| E/I | 深い内向型 | 両向型 | 深い外向型 |
| S/N | 強い感覚型 | バランス | 強い直観型 |
| T/F | 強い思考型 | バランス | 強い感情型 |
| J/P | 強い判断型 | バランス | 強い知覚型 |

### 確率的表現

T=70であっても、アバターは約30%の確率で共感的な反応を見せます。これにより性格の硬直性を防ぎ、自然で人間らしい変動を生み出します。

### 3段階MBTI分析パイプライン

MBTI進化は記憶圧縮時に発生します：

1. **ブラインド分析** — LLMが現在のスコアを知らずにアバターの最近の応答を分析
2. **デルタ計算** — ブラインド結果と現在のスペクトラムの差分
3. **加重適用** — L1→L2およびL2→L3遷移時に設定可能な重みでデルタを適用、極端値では抵抗

## ユーザー性格推論

アバターは自分の性格だけを進化させるのではありません — **あなたの性格も学習します**。

各圧縮サイクルで、ユーザーメッセージを個別に分析してMBTI傾向を推論します。これは複数のサイクルにわたって蓄積されます：

| 信頼度 | アバターの行動 |
|--------|--------------|
| 初期（1-2回の分析） | 未知 — さまざまな質問であなたについて学習 |
| 浮上（2-3回の分析） | 微妙な適応 — 言及せずにコミュニケーションスタイルを調整 |
| 確立（3回以上の分析） | 明示的なパーソナライゼーション — 「直接的な回答がお好みと知っているので...」 |

推論されたユーザー性格はアバターのシステムプロンプトに注入され、週単位のインタラクションを通じて改善される真のパーソナライズされた会話を可能にします。

## 記憶システム — 3層

- **L1（生）**: 設定可能なメッセージ上限と自動圧縮トリガー
- **L2（イベント）**: 関連度スコアは時間とともに減衰；再言及で関連度上昇；低関連度のイベントはL3へ移行
- **L3（キーワード）**: キーワードとして抽出。「よく覚えてないけど…」と不確かに発話
